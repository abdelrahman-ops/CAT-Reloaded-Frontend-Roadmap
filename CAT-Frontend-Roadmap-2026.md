# âš™ï¸ Phase 2: The Core Engineering

> "Frameworks come and go. JavaScript is forever."

## A. JavaScript (The Beast)
*   **Playlist:** [Elzero JavaScript 2021](https://youtube.com/playlist?list=PLDoPjvoNmBAx3kiplQR_oeDqLDBUDYwVv)
*   **Key Modules:**
    1.  **Data Types & Variables:** logic building blocks.
    2.  **DOM Manipulation:** interacting with the page.
    3.  **H.O.F (Map, Filter, Reduce):** data processing.
    4.  **Async/Await & Promises:** fetching data from servers.
    5.  **OOP & Classes:** organizing complex code.

## B. Modern Tooling
You cannot work professionally without these.
*   **Task Runners:** NPM & Scripts.
*   **Bundlers:** Webpack (Understanding the concept).
*   **Preprocessing:** SASS (Writing better CSS).

## C. TypeScript (The Safety Net)
JavaScript with superpowers. Mandatory for large scale apps.
*   **Resources:**
    *   [Elzero TypeScript](https://www.youtube.com/watch?v=yUndnE-2osg&list=PLDoPjvoNmBAy532K9M_fjiAmrJ0gkCyLJ)
*   **Goal:** strict typing, interfaces, and catching bugs before running code.
